## How to use?

### Step 1: Clone this project

```sh
$ git clone https://github.com/cuongw/node-rest-boilerplate.git
```

### Step 2: Remove `.git` folder

```sh
$ cd node-rest-boilerplate && rm -rf .git
```

### Step 3: Install dependencies

```sh
$ yarn
```

### Step 4: Run your project

```sh
$ yarn start
```
